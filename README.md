## Description

Meeting Room booking was done as part of project for web programming course @ The university of Texas at Dallas.The goal of project was to help office employees to book conference rooms online.Employee book,update and cancel conference rooms according his needs. And admin has special permissions to add rooms and delete the bookings.

## How To Use

#### Installation

- Clone the repositry to your local machine and open the cloned repositry in VS Code or any other.
- copy the Mysql script in folder and run it in Mysql.
- Assign databse variable like in route folder files ex. url,db name, password.
- run the below command terminal to start Node.js server:
    ```
      npm start
    ```
